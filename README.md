

# Dungeons & Dragons (low cost) — README

### 1. Как собрать и запустить (на macOS/Linux)
Перейдите в папку src. В ней находятся все `.h`, `.cpp` файлы и `Makefile`.


1. **После загрузки репозитория необходимо зайди в src**:
   ```bash
   cd ./src
   ```
   
2.  **Очистка и сборка**:
    ```bash
    make clean && make
    ```
    
3.  **Запуск**:
    ```bash
    ./game
    ```

---

### 2. Управление (Controls)
Игра работает в пошаговом режиме. После действий игрока враги делают ответный ход.

*   **[W, A, S, D]** — Перемещение игрока. Ход тратится.
*   **[F]** — Атака ближним/дальним боем. Ход тратится.
*   **[M]** — Переключение режима боя (MELEE — больше урона, RANGED — больше дистанция). Не ход.
*   **[1, 2, 3]** — Использовать заклинание из руки. После нажатия нужно выбрать направление (WASD). Ход тратится только при успешном касте.
*   **[B]** — Купить случайное заклинание (цена: 50 очков). Ход тратится только при успешной покупке.

*   **[O]** — Сохранить игру в файл `save.dat` (бинарный формат).
*   **[L]** — Загрузить игру из файла (бинарный формат).
*   **[Q]** — Выход в главное меню.

---

### 3. Игровой цикл 
Логика приложения работает по строгому циклу **Render -> Input -> Update**:

1.  **Render (Отрисовка)**: Консоль очищается, выводится карта уровня, статы игрока (HP, Score), список врагов и лог последних 5 событий.
2.  **Input (Ввод)**: Программа ждет нажатия клавиши.
3.  **Update (Обновление)**: 
    *   Если действие игрока засчитано как «ход», все враги `E` делают шаг в сторону игрока.
    *   Вражеские башни `T` проверяют радиус и стреляют в игрока.
    *   Если враг наступил на ловушку `^`, он получает урон.
    *   Проверяется смерть врагов (начисление очков) и смерть игрока.

---

### 5. Легенда карты
*   `P` — Игрок.
*   `E1, E2...` — Враги.
*   `T` — Вражеская башня.
*   `^` — Установленная ловушка.
*   `#` — Непроходимая стена.
*   `.` — Пустая клетка (пол).
